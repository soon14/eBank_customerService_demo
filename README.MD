命名规范：
- 包名
  - 顶级包名：cn.com.zybank.pe.ebank
  - 微服务root包名： 
    - {顶级包名}.微服务名； 
    - 微服务名中多个单词用下划线分割
    - 微服务包下有跨聚合的类，如 main class, configuration, 跨聚合的服务，跨聚合的查询（连接多个聚合）
  - 聚合名：{微服务root包名}.聚合名
  
  
  
核心组件：
- 模型： 聚合根、实体、值对象
  - 实体：表
  - 值对象： 实体中的属性
  - 聚合：若干实体和值对象的小组
  - 聚合根：小组的管理者，聚合外部的对象只能操作聚合根
  - 辅助的类型: 代表请求数据的 Parameter、代表返回数据的 Result 
- 应用服务：
  - 应用服务的方法对应一个用例、一个请求、一次交互、一次事务
  - 应用服务操作聚合根和基设施
- 资源库：
  - 保存或取出模型对象
- Controller
  - 提供 rest 接口，调用应用服务
